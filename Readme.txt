Project Overview

This project is an AI-powered Gmail client that enables users to manage emails using natural language.

Implemented Features

• Google OAuth 2.0 Authentication
• Gmail API Integration
• Inbox
• Sent Mail
• Email Detail View
• Compose Email
• AI Assistant
• AI Email Summarization
• AI Reply Generation
• AI Email Draft Generation
• Email Search
• Quick AI Commands
• Responsive UI
• Dark Mode
• Toast Notifications
